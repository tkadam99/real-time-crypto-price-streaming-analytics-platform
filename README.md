# real-time-crypto-price-streaming-analytics-platform
A real-time analytics platform that ingests live cryptocurrency price data, streams it through Kafka, processes it for trends and anomaly detection, stores results in a database, and displays live insights on a dashboard.


Project Name

Real-Time Crypto Price Streaming Analytics Platform

Problem Statement

Traditional dashboards often rely on static or delayed data. This project addresses the need for real-time analytics by building a streaming pipeline that ingests live cryptocurrency market updates, processes them asynchronously, detects unusual price movements, stores structured results, and visualizes insights through a web dashboard.

Goal

To design and implement a lightweight but production-style event-driven system that demonstrates streaming architecture, real-time processing, backend integration, and frontend visualization.
